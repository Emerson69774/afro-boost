# Afro Boost

## Description
Afro Boost est un MVP de plateforme pour commander des packs d’actions sociales (likes, vues, partages) sur TikTok, Facebook, Instagram et YouTube.

- Les clients peuvent s’inscrire et commander des packs.
- Les freelances peuvent s’inscrire et exécuter les tâches pour gagner de l’argent.
- Les freelances donnent leur accord pour l’utilisation de leur profil pour actions personnalisées.
- Wallet interne pour clients et freelances.

## Installation
1. Cloner le dépôt.
2. Installer Node.js.
3. Lancer `npm install` pour installer les dépendances.
4. Lancer `node app.js` pour démarrer le serveur.

## Déploiement
Peut être déployé sur Railway ou tout hébergeur Node.js.
